知识点要点

  • computed 缓存依赖；methods 每次必执行；同名属性 vs 方法书写方式不同（{{ x }} vs {{ x() }}）。
  • 局部组件 components: { Foo, Bar } 是 ES2015 简写，等价于 { Foo: Foo, Bar: Bar }。
  • 子组件中若想用其他局部组件，必须在子组件的 components 里再次声明。
