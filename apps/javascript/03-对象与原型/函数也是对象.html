函数也是对象

下方为 console.log 同步输出；也可打开 DevTools Console 查看。

知识点要点

  • 所有函数都是 Function 类型的实例：fn.__proto__ === Function.prototype。
  • Function.prototype 自己也是 Object 实例：Function.prototype.__proto__ === Object.prototype。
  • 任何函数 instanceof Object/Function 都为 true。
  • 所有函数都是 Function 类型的实例：fn.__proto__ === Function.prototype（补充）。
