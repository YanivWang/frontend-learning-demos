盒模型：content-box vs border-box

两个盒子都写了 width: 180px; padding: 24px; border: 8px。

content-box

实际占宽 = 180 + 48 + 16 = 244px

border-box

实际占宽 = 180px，内容区被压缩

margin 折叠

普通父子块中，子元素的垂直 margin 可能和父元素外边距折叠；创建 BFC 后就不会折叠。

普通容器：上下 margin 可能折叠到父级外面

BFC 容器：display: flow-root 阻止 margin 折叠
