用 localforage.iterate 估算剩余容量

请打开 DevTools Console 查看输出。

知识点要点

  • localforage.iterate 遍历所有键值对，每条调用回调。
  • 单条占用 ≈ key.length + JSON.stringify(value).length（字符数近似估算）。
  • iterate 返回 Promise，配合 await 写成顺序代码，便于读。
