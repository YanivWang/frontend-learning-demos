arguments 类数组对象

每个普通函数内部隐式存在 arguments，存储调用时的所有实参。

知识点要点

  • arguments 是每个普通函数内部的类数组对象，按索引存全部实参，有 length。
  • 形参与 arguments 在首次传入时共享同一值；严格模式下不再联动。
  • 没有 map/filter 等数组方法，需转成真数组：Array.prototype.slice.call(arguments)、Array.from、[...arguments]。
  • 箭头函数没有自己的 arguments，会沿作用域链找外层函数的 arguments。
  • ES6 推荐剩余参数 function fn(a, ...rest) {}，rest 是真数组。

面试考点

  • arguments 是什么？ 函数实参的类数组集合。仅普通函数有；严格模式下 arguments.callee 禁用。
  • 如何转数组？ slice.call、Array.from、展开运算符三选一；剩余参数 ...rest 可直接当数组用。
  • 箭头函数有 arguments 吗？ 没有自己的；若内层箭头在外层普通函数里，访问的是外层的 arguments。
  • arguments vs ...rest？ rest 只收集剩余形参之后的多余实参，且是真数组；arguments 包含全部实参但是类数组。新代码优先 rest。
