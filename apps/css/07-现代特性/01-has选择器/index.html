:has() — 根据子节点状态匹配父元素

在浏览器中打开本页，结合下方演示区观察效果；要点与面试答法见上方复习区。

知识点要点

  • :has() 根据子元素或后续兄弟是否存在来匹配祖先/自身，俗称「父选择器」。
  • 卡片 :has(input:checked) 高亮选中项，无需 JS 切换 class。
  • form :has(:invalid) 可在存在无效表单项时提示提交区。
  • 浏览器支持已较广（Chrome 105+、Safari 15.4+、Firefox 121+），降级时样式不生效但不影响布局。
