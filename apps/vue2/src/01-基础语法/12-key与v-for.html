key 复用控制 + v-for 遍历数组/对象

在浏览器中操作下方交互演示；复习要点与面试答法见上方区块。

知识点要点

  • Vue 复用 input 等元素会保留旧值；给元素加 :key 可强制让其被「视为不同元素」。
  • v-for 既能遍历数组 (item, index)，也能遍历对象 (value, key, index)。
  • v-for 遍历数组/对象时建议写 :key（稳定 id）；本页同时演示「无 key 复用」与「有 key 强制重建」。
