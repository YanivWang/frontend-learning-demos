Vue3 实现可交互的 Tooltip

在浏览器中打开本页，结合下方演示区观察效果；要点与面试答法见上方复习区。

知识点要点

  • 使用 mouseenter/mouseleave 触发显示/隐藏；移入 tooltip 时清除隐藏定时器。
  • 100ms 的延迟隐藏，给鼠标从触发区移动到 tooltip 留出过渡时间。
  • tooltip 用绝对定位 + ::after 三角实现指向尖角。
