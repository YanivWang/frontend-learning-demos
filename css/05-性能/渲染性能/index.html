CSS 渲染性能

动画优先 transform / opacity

will-change：短期提示浏览器优化

悬停我（:hover 时设置 will-change: transform）

只在即将动画前声明 will-change，动画结束后移除，避免长期占用显存。

面试回答框架

容易重排：width、height、margin、padding、top、left、font-size
容易重绘：color、background、box-shadow
常用于高性能动画：transform、opacity

隔离影响范围

contain: layout paint 可把布局和绘制影响限制在局部，适合复杂组件谨慎使用。
