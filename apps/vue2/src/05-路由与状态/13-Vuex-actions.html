Vuex actions 详解（载荷 / 异步 / Promise / 嵌套 dispatch）

在浏览器中操作下方交互演示；复习要点与面试答法见上方区块。

知识点要点

  • mutation 必须同步；action 可以包含异步逻辑，最后 commit 一个 mutation。
  • dispatch 也返回 Promise，可在组件里用 await 等异步流程。
  • state 需先声明 count 等字段，否则 mutation 自增会得到 NaN。
