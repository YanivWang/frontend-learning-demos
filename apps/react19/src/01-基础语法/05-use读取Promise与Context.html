use() 读取 Promise 与 Context

在浏览器中操作下方交互演示；复习要点与面试答法见上方区块。

知识点要点

  • use(usable)：在 render 中读取 Promise 或 Context（React 19 新 Hook）。
  • 读取 Promise 时组件会挂起，需外层 Suspense 提供 fallback。
  • 可在条件分支中调用 use()（与 useState 等 Hook 规则不同）。
  • 读取 Context 时可直接 use(Context)，等价于 useContext。
