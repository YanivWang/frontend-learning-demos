纯 CSS tooltip（上下左右四方向）

在浏览器中打开本页，结合下方演示区观察效果；要点与面试答法见上方复习区。

知识点要点

  • ::before 画三角；::after 用 attr(data-tip) 拿提示文字。
  • 默认 opacity:0；:hover 时 opacity:1 + 反向移动实现「出现 + 弹出」动效。
  • transparent border + border-XXX-color 是经典「画三角形」技巧。
