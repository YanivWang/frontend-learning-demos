FileReader 读取文件并存入 localforage

请打开 DevTools Console 查看输出。

知识点要点

  • readAsDataURL 把文件读成 base64 字符串；下方用 result.length×2 粗算 UTF-16 字符占用。
  • 大文件直接读 dataURL 内存吃紧，更适合 readAsArrayBuffer / Blob 存储。
  • 始终监听 onloadend / onerror，便于在异常时给出反馈。
