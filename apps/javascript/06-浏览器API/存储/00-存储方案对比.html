浏览器存储方案对比

在浏览器中打开本页，结合下方演示区观察效果；要点与面试答法见上方复习区。

知识点要点

  • Cookie 会随同源请求自动发送，适合小型认证信息，但容量小。
  • localStorage/sessionStorage 是同步 API，容量较大但可能阻塞主线程。
  • IndexedDB 是异步数据库，适合大数据和文件，localforage 是常用封装。

面试考点

  • Cookie 会随同源请求自动发送？ Cookie 会随同源请求自动发送，适合小型认证信息，但容量小
