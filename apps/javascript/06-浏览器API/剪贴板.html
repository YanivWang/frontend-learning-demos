把文本复制到剪贴板

在浏览器中打开本页，结合下方演示区观察效果；要点与面试答法见上方复习区。

知识点要点

  • 老方案 document.execCommand('copy')：兼容性最好但已废弃。
  • 新方案 navigator.clipboard.writeText：需 HTTPS + 用户手势触发。
  • 易错点：必须在用户点击事件回调里调用，否则浏览器会拒绝。
