React.lazy 与 Suspense

在浏览器中操作下方交互演示；复习要点与面试答法见上方区块。

知识点要点

  • React.lazy 用于动态加载组件，常配合路由或大模块做代码分割。
  • Suspense 的 fallback 会在懒加载组件未就绪时展示。
  • 真实项目里 lazy 通常写成 lazy(() => import('./Page.jsx'))。
  • 本静态 HTML 无打包环境，用 Promise 模拟动态加载。
