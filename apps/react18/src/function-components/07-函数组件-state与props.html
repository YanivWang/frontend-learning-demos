函数组件 state 与 props 综合

在浏览器中操作下方交互演示；复习要点与面试答法见上方区块。

知识点要点

  • props 由父组件传入，只读；state 由组件自身 useState 管理。
  • 父改 props 会触发子组件重渲染；子改 state 只影响自身。
  • 受控于父的展示用 props，交互产生的局部状态用 state。
  • 对比 class 02：函数组件用 useState 替代 this.state / this.setState。
