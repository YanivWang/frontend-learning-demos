JSX 表达式与 React 19 自动批处理

在浏览器中操作下方交互演示；复习要点与面试答法见上方区块。

知识点要点

  • JSX 花括号内可写任意 JavaScript 表达式（变量、三元、函数调用）。
  • 注意 && 短路：0、空字符串可能被渲染到 DOM。
  • React 19 在 createRoot 下对 setTimeout、Promise 等中的 setState 默认批处理。
  • 需要同步读 DOM 时仍可用 flushSync 退出批处理。
