自定义组件上 v-bind:class 行为

在浏览器中操作下方交互演示；复习要点与面试答法见上方区块。

知识点要点

  • 在自定义组件上写 :class，最终会合并到「组件根节点」的 class 属性。
  • 与父模板原有的 class 是合并而非覆盖，多个 class 都会出现。
  • 多根节点组件（Vue3）的 class 透传规则不同，注意区分。
