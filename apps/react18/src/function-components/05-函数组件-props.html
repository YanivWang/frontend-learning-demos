知识点要点

  • 函数组件唯一参数是 props 对象；从父级传入的所有 attribute 都被打包进 props。
  • props 是只读的；要更新视图需重新调用 render（或者使用 state / hooks）。
  • createRoot 只创建一次后复用；ReactDOM.render 已废弃。
