class 声明的关键特性

知识点要点

  • 类声明不提升，存在暂时性死区；class 主体默认严格模式。
  • 静态字段在类初始化时执行，实例字段仅在 new 时初始化。
  • 易错点：无 extends 的构造器不能调用 super()；class 名标识符不能重新声明。
