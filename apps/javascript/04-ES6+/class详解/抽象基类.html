用 new.target 实现抽象基类

下方为 console.log 同步输出；也可打开 DevTools Console 查看。

知识点要点

  • new.target 指向直接被 new 的构造器；直接 new 抽象类时等于其本身。
  • 通过 if (new.target === Animal) throw 阻止直接实例化抽象基类。
  • 派生类 constructor 必须先调用 super() 才能使用 this。
