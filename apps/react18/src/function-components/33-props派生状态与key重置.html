props 派生状态与 key 重置

在浏览器中操作下方交互演示；复习要点与面试答法见上方区块。

知识点要点

  • 避免 props 变化时用 useEffect 同步到 state（多余且易出 bug）。
  • 能直接算的值在 render 里派生；需要重置整块 UI 时用 key。
  • 对比 class 09 getDerivedStateFromProps：函数组件优先「派生计算」或 key 重置。
  • 必须保留本地编辑态时，才在 props 变化时有选择地重置 state。
