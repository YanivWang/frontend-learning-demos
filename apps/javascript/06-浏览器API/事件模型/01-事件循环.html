事件循环综合（同步 + Promise + async/await + setTimeout）

请打开 DevTools Console 查看输出。

知识点要点

  • 同步任务先执行；微任务（Promise.then、await 之后）优先于宏任务。
  • await 之后的代码相当于 .then 回调，属于微任务。
  • setTimeout 回调是宏任务，会在当前微任务批次执行完后才运行。
  • 同步任务先执行；微任务（Promise.then、await 之后）优先于宏任务（补充）。
