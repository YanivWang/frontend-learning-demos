View Transitions

知识点要点

  • document.startViewTransition() 触发默认交叉淡入淡出。
  • ::view-transition-old / ::view-transition-new 可自定义动画。
  • 适合 SPA 路由切换、主题切换、列表 reorder。
  • 不支持时 @supports 降级为普通 transition。
