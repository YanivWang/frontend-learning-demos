defaultProps、静态属性与 displayName

在浏览器中操作下方交互演示；复习要点与面试答法见上方区块。

知识点要点

  • Component.defaultProps 为未传 props 提供默认值（React 18 仍可用，新代码更推荐参数默认值）。
  • static 静态属性挂在类上，实例通过 constructor 访问不到。
  • displayName 便于 DevTools 调试；HOC 包装时尤其有用。
