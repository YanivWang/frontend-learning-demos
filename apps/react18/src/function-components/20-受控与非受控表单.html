受控组件与非受控组件

在浏览器中操作下方交互演示；复习要点与面试答法见上方区块。

知识点要点

  • 受控组件: 表单值由 React state 控制，value + onChange 成对出现。
  • 非受控组件: 表单值保存在 DOM 中，通过 ref 在需要时读取。
  • 业务表单、校验、联动通常用受控组件；简单一次性读取可用非受控组件。
