知识点要点

  • 通过四个同心圆环 + 不同 animation-delay 形成「追逐」效果。
  • border-color: c transparent transparent transparent 只保留顶部一段，旋转后看似一条弧。
  • cubic-bezier(0.5,0,0.5,1) 是定制的缓动曲线。
