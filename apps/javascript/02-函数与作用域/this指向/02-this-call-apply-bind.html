显式绑定 this：call / apply / bind

请打开 DevTools Console 查看输出。

知识点要点

  • call(thisArg, arg1, arg2, ...)；apply(thisArg, [args])。
  • bind 返回一个绑定了 this 的新函数，不会立即执行。
  • 易错点：apply 第三个参数会被忽略；apply(null) 严格模式下 this=null，非严格为 window。
  • call(thisArg, arg1, arg2, ...)；apply(thisArg, [args])（补充）。
