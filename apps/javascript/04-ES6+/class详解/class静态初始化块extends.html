静态初始化块在继承场景的特性

下方为 console.log 同步输出；也可打开 DevTools Console 查看。

知识点要点

  • 父类静态先初始化，子类静态后初始化。
  • 子类静态块中可用 super.xxx 访问父类静态属性/方法。
  • 静态方法在类本身上，需要通过类名调用。
