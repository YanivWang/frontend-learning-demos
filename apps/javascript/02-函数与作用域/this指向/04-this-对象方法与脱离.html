方法被「赋值出去」后 this 的脱离

请打开 DevTools Console 查看输出。

知识点要点

  • 通过 obj.fn() 调用，this 指向 obj。
  • 将方法引用赋给变量后再调用（funA()），this 退化为 window/undefined。
  • 解决：调用前 .bind(obj) 或使用箭头函数 () => obj.fn()。
  • 通过 obj.fn() 调用，this 指向 obj（补充）。

面试考点

  • 通过 obj.fn() 调用？ 通过 obj.fn() 调用，this 指向 obj
  • 将方法引用赋给变量后再调用（funA()）？ 将方法引用赋给变量后再调用（funA()），this 退化为 window/undefined
  • 方法被「赋值出去」后 this 的脱离的核心概念是什么？ 通过 obj.fn() 调用，this 指向 obj；将方法引用赋给变量后再调用（funA()），this 退化为 window/undefined
