class 组件 refs 访问 DOM 与实例

在浏览器中操作下方交互演示；复习要点与面试答法见上方区块。

知识点要点

  • React.createRef() 创建 ref，挂载后 ref.current 指向 DOM 或 class 实例。
  • 回调 ref ref={(el) => ...} 在挂载/卸载时分别被调用。
  • ref 改变不会触发 render；适合聚焦、测量 DOM、命令式操作。
  • 字符串 ref 已废弃；函数组件 ref 需 forwardRef（见 function-components/27）。
