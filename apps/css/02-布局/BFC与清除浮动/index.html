在浏览器中打开本页，结合下方演示区观察效果；要点与面试答法见上方复习区。

知识点要点

  • BFC 是独立布局区域，内部浮动会参与高度计算。
  • 常见触发: display: flow-root、overflow:hidden、float、position:absolute/fixed、inline-block。
  • BFC 可用于清除浮动、阻止 margin 折叠、避免文字环绕浮动元素。
  • BFC 是独立布局区域，内部浮动会参与高度计算（补充）。
