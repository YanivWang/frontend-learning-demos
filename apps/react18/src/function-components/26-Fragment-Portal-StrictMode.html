Fragment、Portal 与 StrictMode

在浏览器中操作下方交互演示；复习要点与面试答法见上方区块。

知识点要点

  • Fragment 用于返回多个兄弟节点，不额外创建 DOM 包裹层。
  • Portal 可以把子节点渲染到父组件 DOM 层级之外，常用于弹窗、浮层。
  • StrictMode 只在开发环境生效，用来暴露副作用、废弃 API 等问题。
