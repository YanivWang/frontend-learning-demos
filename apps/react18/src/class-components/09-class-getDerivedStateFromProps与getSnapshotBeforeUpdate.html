getDerivedStateFromProps 与 getSnapshotBeforeUpdate

在浏览器中操作下方交互演示；复习要点与面试答法见上方区块。

知识点要点

  • getDerivedStateFromProps(props, state): 渲染前根据 props 派生 state，必须纯函数、无权访问 this。
  • 常见用途: 受控组件模式、props 变化时同步内部 state（慎用，易冗余 state）。
  • getSnapshotBeforeUpdate(prevProps, prevState): DOM 更新前调用，返回值传给 componentDidUpdate 第三参数。
  • 常见用途: 滚动位置恢复、聊天窗口保持在底部。
