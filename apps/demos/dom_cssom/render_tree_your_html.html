DOM + CSSOM → Render Tree（只留可见节点）
DOM
  • html
    • <head> 整块丢弃
    • body
      • h1
        • "DOM"
      • h3
        • "DOM讲解"
      • <script> 丢弃
Render Tree（RenderObject）
  • RenderView（根）
    • RenderBlock — <body> display: block
      margin: 8px （浏览器默认）
      • RenderBlock — <h1> display: block
        color: blue ←你写的
        font-size: 2em （UA默认）
        font-weight: bold （UA默认）
        margin: .67em 0 （UA默认）
        • RenderText — "DOM" 继承 color: blue
      • RenderBlock — <h3> display: block
        color: red ←你写的
        font-size: 1.17em （UA默认）
        font-weight: bold （UA默认）
        margin: 1em 0 （UA默认）
        • RenderText — "DOM讲解" 继承 color: red
被排除的节点（不进 Render Tree）：
• <head> 及其所有子节点（meta / title / style）— 从不渲染
• <script> — 无视觉输出
• 任何 display: none 的节点（你的 HTML 里没有，但这是最常见的排除原因）
• visibility: hidden 的节点 会留在 Render Tree（占位但不绘制）
