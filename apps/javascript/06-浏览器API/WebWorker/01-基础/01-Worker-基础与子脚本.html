Worker 基础与子脚本

请打开 DevTools Console 查看输出。

知识点要点

  • new Worker(path) 启动独立线程；主线程通过 postMessage/onmessage 通信。
  • 数据采用结构化克隆传递，函数 / DOM 节点不可传。
  • 用 worker.terminate() 主动结束；子线程内部用 self.close()。
