box-shadow vs filter: drop-shadow 对比

在浏览器中打开本页，结合下方演示区观察效果；要点与面试答法见上方复习区。

知识点要点

  • box-shadow 沿元素矩形外边产生阴影，无视虚线/不透明像素。
  • filter: drop-shadow 沿非透明像素轮廓产生阴影，对 PNG/SVG 描边更友好。
  • drop-shadow 使用 GPU 合成，但对触发位图层的元素会增大显存。
