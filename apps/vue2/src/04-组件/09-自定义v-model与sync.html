知识点要点

  • Vue2 自定义组件 v-model 默认使用 prop: value 与 event: input。
  • 组件可用 model 选项改成其它 prop/event，比如 checked/change。
  • .sync 是 :prop + @update:prop 的语法糖，适合子组件请求父组件更新某个 prop。
  • Vue2 自定义组件 v-model 默认使用 prop: value 与 event: input（补充）。
