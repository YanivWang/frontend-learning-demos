面试回答

  • Vue 3.5 之前：<script setup> 里 const { x } = defineProps() 也会丢响应，需 toRefs 或直接用 props.x。
  • Vue 3.5+：编译器会把 props 解构转成响应式引用，工程里可以直接解构使用。
  • 普通 setup(props)（本仓库全局构建）仍不能直接解构；可用 toRefs(props) 达到同样效果。
