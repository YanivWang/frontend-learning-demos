attrs 与属性透传

默认透传（单根组件自动落到根元素）

保存

inheritAttrs: false + 手动 v-bind="$attrs"

取消

{{ message }}

面试回答

  • attrs 适合做二次封装组件，比如按钮、输入框、表格组件。
  • 单根组件默认会把未声明为 props 的属性透传到根元素；关闭后需手动 v-bind="$attrs"。
  • Vue3 中 class、style、普通属性和事件监听都可作为透传属性处理。
  • 多根组件不会自动决定透传到哪里，需要显式 v-bind="$attrs"。
