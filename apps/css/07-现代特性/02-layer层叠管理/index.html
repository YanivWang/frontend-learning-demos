@layer — 分层管理样式优先级

在浏览器中打开本页，结合下方演示区观察效果；要点与面试答法见上方复习区。

知识点要点

  • @layer 把样式分组，未分层样式优先级高于任意 @layer 内规则。
  • 同层内仍按选择器特异性 + 源码顺序决定；层之间按声明顺序：后声明的层覆盖先声明的层。
  • 适合第三方 reset、组件库、工具类分层，避免 !important 滥用。
  • 可 @layer utilities { ... } 或在文件顶部 @layer base, components, utilities;。
