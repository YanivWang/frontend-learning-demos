表头固定 + 表体滚动

在浏览器中打开本页，结合下方演示区观察效果；要点与面试答法见上方复习区。

知识点要点

  • 方案 A：用 display: table/block 切换 + table-layout 固定列宽，需手动扣滚动条宽度。
  • 方案 B：thead th { position: sticky; top: 0 }，无需切换 display，保留列对齐。
  • sticky 必须依附「可滚动祖先」，所以外层 wrap 需 overflow:auto。

面试考点

  • sticky 必须依附「可滚动祖先」？ 方案 B：thead th { position: sticky; top: 0 }，无需切换 display，保留列对齐；sticky 必须依附「可滚动祖先」，所以外层 wrap 需 overflow:auto
  • 表头固定 + 表体滚动的核心概念是什么？ 方案 A：用 display: table/block 切换 + table-layout 固定列宽，需手动扣滚动条宽度；方案 B：thead th { position: sticky; top: 0 }，无需切换 display，保留列对齐
  • 表头固定 + 表体滚动在实际项目里怎么用？ 方案 A：用 display: table/block 切换 + table-layout 固定列宽，需手动扣滚动条宽度；方案 B：thead th { position: sticky; top: 0 }，无需切换 display，保留列对齐
