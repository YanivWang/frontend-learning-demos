new 调用构造函数时 this 的行为

下方为 console.log 同步输出；也可打开 DevTools Console 查看。

知识点要点

  • new 构造时 this 指向新创建的实例，绝不会污染外层作用域。
  • 不加 new 直接调用，this 退化为 window（非严格）或 undefined（严格）。
  • 推荐为构造函数加首字母大写规范，避免漏写 new。
  • new 构造时 this 指向新创建的实例，绝不会污染外层作用域（补充）。
