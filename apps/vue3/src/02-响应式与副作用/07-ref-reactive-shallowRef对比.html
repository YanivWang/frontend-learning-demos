ref / shallowRef / reactive 三种响应式 API 对比

在浏览器中操作下方交互演示；复习要点与面试答法见上方区块。

知识点要点

  • ref 返回 { value: ... } 包装；JS 中读写必须 .value，模板中自动解包。
  • reactive 返回的是「Proxy 响应式对象」，访问字段无 .value，但解构会丢响应。
  • shallowRef 只对 .value 的赋值「整体替换」做响应，内部嵌套字段不再追踪。
  • 须从 Vue 解构 reactive；JS 中访问 ref 必须 .value（模板中自动解包）。
