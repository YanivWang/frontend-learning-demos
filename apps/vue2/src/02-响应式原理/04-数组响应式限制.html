Vue2 数组响应式限制

在浏览器中操作下方交互演示；复习要点与面试答法见上方区块。

知识点要点

  • Vue2 通过重写数组的 7 个变更方法触发更新：push / pop / shift / unshift / splice / sort / reverse。
  • 通过索引直接赋值 arr[index] = value 不会触发视图更新。
  • 直接修改 arr.length 也不会触发视图更新。
  • 修复方式：Vue.set(arr, index, value) 或 arr.splice(index, 1, value)。
