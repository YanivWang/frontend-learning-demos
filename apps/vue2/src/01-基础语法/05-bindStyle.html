:style 三种写法（行内对象 / 对象变量 / 数组）

在浏览器中操作下方交互演示；复习要点与面试答法见上方区块。

知识点要点

  • 行内对象写法可直接引用 data 字段：{color: activeColor}。
  • 数组写法可同时合并多个样式对象（后者覆盖前者）。
  • CSS 属性名既能用驼峰 fontSize，也能用串 'font-size'，但同 key 不要混写。
