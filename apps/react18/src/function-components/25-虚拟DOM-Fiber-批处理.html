虚拟 DOM、Fiber 与 React 18 自动批处理

在浏览器中操作下方交互演示；复习要点与面试答法见上方区块。

知识点要点

  • 虚拟 DOM 是描述 UI 的普通 JS 对象，React 用它计算真实 DOM 更新。
  • Fiber 是 React 的内部节点结构，让渲染工作可拆分、可中断、可恢复。
  • React 18 createRoot 下，事件、定时器、Promise 等异步回调中的状态更新也会自动批处理。
  • 面试不必手写 Fiber，但要能说清楚它解决了大更新阻塞主线程的问题。
