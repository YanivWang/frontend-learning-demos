用静态私有标志位模拟「私有构造函数」

请打开 DevTools Console 查看输出。

知识点要点

  • JS 没有真正的 private constructor，借助静态私有字段 + 工厂方法模拟。
  • 直接 new PrivateConstructor() 会抛错；只能通过 PrivateConstructor.create()。
  • 私有静态字段不能在类外部访问。
