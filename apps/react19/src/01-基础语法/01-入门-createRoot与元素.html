React 19 元素与 createRoot 渲染

在浏览器中操作下方交互演示；复习要点与面试答法见上方区块。

知识点要点

  • JSX 经 Babel 转译为 React.createElement，得到 React 元素（普通对象）。
  • ReactDOM.createRoot(container).render(element) 为唯一推荐挂载方式。
  • React 19 仍基于元素不可变：更新界面需 render 新元素树。
  • 本模块使用 React 19 运行时（umd-react），与 apps/react18 的 18.3.1 分离。
