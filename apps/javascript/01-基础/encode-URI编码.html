URI 编码 API（encodeURI / encodeURIComponent / decode）

请打开 DevTools Console 查看输出。

知识点要点

  • encodeURI：编码完整 URL，保留 : / ? # 等 URI 结构字符，只编码中文等特殊字符。
  • encodeURIComponent：编码 URL 组件（query 值、path 段），会编码 / ? & = 等。
  • 解码用 decodeURI / decodeURIComponent； malformed 序列抛 URIError。
  • 拼接 URL：?q=${encodeURIComponent(keyword)}，不要对整个 URL 用 encodeURIComponent。
  • escape 已废弃，编码规则与 UTF-8 百分号编码不一致。

面试考点

  • encodeURI vs encodeURIComponent？ 前者保 URI 结构；后者适合 query/path 片段。对整个 URL 误用 Component 会把 :// 也编码。
  • 什么时候用哪个？ 整链 encodeURI；单个参数 encodeURIComponent；现代也可用 URLSearchParams。
  • escape 为何废弃？ 非标准、不支持 Unicode 完整范围；统一用 encodeURIComponent/decodeURIComponent。
