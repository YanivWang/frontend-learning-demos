Clock 演示 setState 批处理（对象式 vs 函数式）

在浏览器中操作下方交互演示；复习要点与面试答法见上方区块。

知识点要点

  • counterOld：三次对象式 setState 都基于「当前 state」+1，合并后只 +1。
  • counterNew：三次函数式 setState 都拿到「最新合并 state」，最终 +3。
  • componentWillUnmount 拼写必须正确，否则 React 不会调用，定时器无法清理。
