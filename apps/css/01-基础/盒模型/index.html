盒模型：content-box vs border-box

在浏览器中打开本页，结合下方演示区观察效果；要点与面试答法见上方复习区。

知识点要点

  • content-box: width/height 只包含内容区，padding 和 border 会额外撑大元素。
  • border-box: width/height 包含 content + padding + border，更适合工程化布局。
  • 垂直 margin 会在普通文档流中折叠；BFC、padding、border、inline-block 等可阻止折叠。
  • content-box: width/height 只包含内容区，padding 和 border 会额外撑大元素（补充）。
