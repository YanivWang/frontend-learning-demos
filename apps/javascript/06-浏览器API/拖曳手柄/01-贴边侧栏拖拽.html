拖拽调整侧栏宽度（Vue3 + requestAnimationFrame 版）

知识点要点

  • mousedown 起点 → mousemove 计算偏移 → mouseup 解绑事件，是拖拽三段经典写法。
  • 拖拽过程中给容器加 .no-transition，避免 transition 让宽度跟手不及时。
  • 用 requestAnimationFrame 合帧节流，宽度变化与浏览器渲染节奏同步。

可调整宽度的悬浮导航栏（优化版）

优化后的版本解决了拖拽卡顿问题，使用了requestAnimationFrame实现流畅动画， 并在拖拽过程中禁用过渡效果，确保操作顺滑。

⚡ 性能优化

使用requestAnimationFrame确保动画与屏幕刷新率同步， 避免拖拽过程中的卡顿现象，提供流畅的用户体验。

🔄 过渡控制

拖拽过程中禁用CSS过渡效果，减少浏览器重排计算， 结束后恢复过渡效果，平衡性能与视觉体验。

🎯 精准控制

优化事件处理逻辑，减少不必要的DOM操作， 确保每次调整都能精准响应鼠标移动。
