ES Module import/export 静态导入与动态导入

请打开 DevTools Console 查看输出。需通过 http(s) 协议访问本页（file:// 无法加载模块）。

知识点要点

  • <script type="module"> 自动启用严格模式，import 必须是顶层语句。
  • 静态导入 import {...} from '...'；动态导入 import('...').then(...) 可在运行时按需加载。
  • 模块在浏览器中需要通过 http(s) 协议加载（直接 file:// 不行）。

面试考点

  • 静态 vs 动态 import？ 静态 import 编译期确定依赖，利于 tree-shaking；动态 import() 返回 Promise，适合懒加载和条件加载。
  • type="module" 有何特性？ 自动严格模式、defer 执行、单例缓存、CORS 限制。
  • 为何 file:// 不行？ 浏览器安全策略要求模块脚本通过 http(s) 加载，本地开发用 dev server。
