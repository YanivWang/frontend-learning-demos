CSRF 防护要点

  • Cookie 设置 SameSite=Lax/Strict，敏感操作用 SameSite=Strict
  • 表单/接口携带不可预测的 CSRF Token（服务端校验）
  • 校验 Origin / Referer 是否来自可信域
// 响应头示例
Content-Security-Policy: default-src 'self'; script-src 'self'
Set-Cookie: session=...; HttpOnly; Secure; SameSite=Lax
