React.memo 与不可变更新

在浏览器中操作下方交互演示；复习要点与面试答法见上方区块。

知识点要点

  • React.memo 会在 props 浅比较相等时跳过函数组件重渲染。
  • 浅比较看引用，数组/对象直接原地修改会破坏 React 的更新判断。
  • 性能优化前先定位问题，不要为了“看起来专业”滥用 memo。
