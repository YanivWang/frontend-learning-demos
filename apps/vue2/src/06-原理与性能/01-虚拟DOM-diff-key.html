虚拟 DOM / diff / key

在浏览器中操作下方交互演示；复习要点与面试答法见上方区块。

知识点要点

  • 虚拟 DOM 是用普通 JS 对象描述真实 DOM，更新时先比较 vnode，再最小化操作真实 DOM。
  • Vue2 diff 以同层比较为主，不做跨层级移动比较。
  • key 用来标识节点身份；稳定且唯一的 key 能帮助 Vue 正确复用和移动节点。
  • 不推荐用数组 index 作为可变列表的 key，插入/删除时容易错误复用组件状态。
