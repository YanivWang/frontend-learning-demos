React 元素的「不可变」特性

在浏览器中操作下方交互演示；复习要点与面试答法见上方区块。

知识点要点

  • React 元素是不可变 JS 对象：要更新视图，只能重新创建并重新 render。
  • React 内部用 diff 找出真实 DOM 的最小差异，只更新需要变化的节点。
  • createRoot 在外面创建一次，每次 tick 复用同一个 root 调用 render（勿每次 tick 重新 createRoot）。
