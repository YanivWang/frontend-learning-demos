数组方法与常见面试题

下方为 console.log 同步输出；也可打开 DevTools Console 查看。

知识点要点

  • 不改原数组：map、filter、reduce、some、every、find、findIndex、slice、concat、flat、includes 等。
  • 会改原数组：push、pop、shift、unshift、splice、sort、reverse、fill、copyWithin。
  • sort() 默认按字符串 Unicode 排序，数字排序需 (a,b)=>a-b。
  • includes(NaN) 为 true；indexOf(NaN) 为 -1（=== 比较）。
  • 手写常考：去重（Set / filter+indexOf）、扁平化（flat / reduce）、分组（reduce）。

面试考点

  • 哪些方法改原数组？ 记「栈队列 splice sort reverse」+ push/pop/shift/unshift。需要不可变数据时对副本操作或 spread。
  • 数组去重？ [...new Set(arr)] 或 filter+indexOf。对象数组按 key 去重需 Map。
  • map vs forEach？ map 返回新数组；forEach 无返回值，不能链式。forEach 里 return 只跳出当前回调。
  • [1,2,3].map(parseInt)？ 得 [1, NaN, NaN]：map 传 (value,index)，parseInt('2',1) 和 parseInt('3',2) 为 NaN。应写 .map(Number) 或显式 radix。
