知识点要点

  • new IntersectionObserver(callback, options)：监听目标元素与视口的相交。
  • threshold 控制触发阈值；rootMargin 可扩大/缩小判定范围（如预加载）。
  • 元素进入视口后再观察 disconnect/unobserve，避免重复触发。
  • new IntersectionObserver(callback, options)：监听目标元素与视口的相交（补充）。
