useRef 保存 DOM 与可变值

在浏览器中操作下方交互演示；复习要点与面试答法见上方区块。

知识点要点

  • useRef 返回稳定对象 { current: ... }。
  • ref.current 改变不会触发组件重新渲染。
  • 常见用途: 访问 DOM、保存定时器 ID、保存不会驱动视图的可变值。
  • useRef 与 useState 的核心区别: state 参与渲染，ref 不参与渲染。
