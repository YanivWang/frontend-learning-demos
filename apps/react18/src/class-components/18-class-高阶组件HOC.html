高阶组件 HOC（Higher-Order Component）

在浏览器中操作下方交互演示；复习要点与面试答法见上方区块。

知识点要点

  • HOC 是函数：接收组件，返回增强后的新组件（不修改原组件）。
  • 常见用途: 注入 props、权限校验、日志、复用生命周期逻辑。
  • 注意: 不要在内层 HOC 使用 render 里的箭头函数包装，以免破坏 ref 与 reconciliation。
  • Hooks 时代很多 HOC 场景可用自定义 Hook 替代，但面试仍常问 HOC 原理。
