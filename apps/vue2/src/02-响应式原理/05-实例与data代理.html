实例 vm 与 _data 的关系

在浏览器中操作下方交互演示；复习要点与面试答法见上方区块。

知识点要点

  • data 中字段会被代理到 vm 上：vm.message === vm._data.message。
  • vm.data 不存在；想拿原始 data 对象用 vm.$data 或 vm._data。
  • mounted 时模板已渲染完毕，但子组件不一定完成挂载。
