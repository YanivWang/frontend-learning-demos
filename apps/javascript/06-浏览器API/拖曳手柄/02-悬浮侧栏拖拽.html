可调整宽度的悬浮导航栏（修复版）

这是一个使用Vue3框架实现的可调整宽度的悬浮左侧导航栏。 已修复拖拽功能无法正常工作的问题，确保用户可以流畅调整导航栏宽度。

🔧 拖拽修复

修复了拖拽功能无法正常工作的问题，确保用户可以流畅调整导航栏宽度。 拖拽范围限制在178px到280px之间。

⚡ 性能优化

使用requestAnimationFrame确保动画与屏幕刷新率同步， 避免拖拽过程中的卡顿现象。

🔄 自动适配

主内容区域的左边距会自动跟随导航栏宽度变化， 确保内容不会被导航栏遮挡。
