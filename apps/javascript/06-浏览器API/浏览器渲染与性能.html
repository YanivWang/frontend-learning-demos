浏览器渲染与性能

在浏览器中打开本页，结合下方演示区观察效果；要点与面试答法见上方复习区。

知识点要点

  • 关键渲染路径：HTML -> DOM，CSS -> CSSOM，然后生成 Render Tree。
  • Layout 计算位置尺寸，Paint 绘制像素，Composite 合成图层。
  • 批量 DOM 操作、读写分离、transform/opacity 动画可以减少性能压力。
