# Vue 2

Vue 2.7 时代的 demo（`libs/vue2.js` 是 2.7.16 版本）。

## 目录

| 目录 | 知识点 |
|---|---|
| `src/响应式原理/` | `Object.defineProperty` 响应式、`Vue.set` 实战、组件基础、timeline |
| `src/基础语法/` | 模板语法、`v-if`/`v-else`、`computed`、`watch`、`bindClass` / `bindStyle`、事件修饰符、`ref`、`key` |
| `src/组件/` | `Prop`、插槽 `slot`、动态/异步组件、`bindComponent`、参数解构 |
| `src/路由与状态/` | `vue-router` 命名 / 嵌套 / 编程式 / 重定向 / 守卫、Vuex `Module` / action |
| `src/源码简读/` | Vue 源码阅读笔记 |
| `src/生命周期.html` | 生命周期演示 |

## 依赖

`libs/vue2.js` `libs/vue-router.js` `libs/vuex.js` `libs/es6-promise.auto.js`，版本见 [`libs/README.md`](libs/README.md)。
