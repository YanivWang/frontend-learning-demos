template 编译到 render 函数

在浏览器中操作下方交互演示；复习要点与面试答法见上方区块。

知识点要点

  • Vue2 完整版运行时包含模板编译器，能把 template 编译成 render 函数。
  • 编译大致流程：template -> AST -> render code -> render function。
  • 运行时只负责执行 render 得到 vnode，再通过 patch 更新真实 DOM。
  • 使用 vue-loader / 构建工具时，单文件组件模板通常在构建阶段预编译。
