class 组件事件 this 绑定的三种方式

在浏览器中操作下方交互演示；复习要点与面试答法见上方区块。

知识点要点

  • 方式一: constructor 中 this.handle = this.handle.bind(this)。
  • 方式二: JSX 箭头函数 onClick={(e) => this.handle(e)}（每次 render 新建函数）。
  • 方式三: 类字段箭头函数 handle = (e) => {}（推荐，this 自动绑定）。
  • 直接 onClick={this.handle} 会导致 this 为 undefined（非严格模式下可能是 window）。
