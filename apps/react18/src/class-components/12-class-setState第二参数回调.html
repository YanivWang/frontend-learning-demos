setState 第二参数回调与 state 读取时机

在浏览器中操作下方交互演示；复习要点与面试答法见上方区块。

知识点要点

  • setState(updater, callback) 的 callback 在 state 提交且 DOM 更新后执行。
  • setState 后立即读 this.state 仍是旧值（异步批处理）。
  • callback 里可读最新 DOM；需要基于新 state 做副作用时用 callback 或 componentDidUpdate。
