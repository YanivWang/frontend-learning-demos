知识点要点

  • useReducer 适合多个状态字段有关联、更新逻辑较集中或需要 action 语义的场景。
  • reducer 必须是纯函数: 相同 state + action 得到相同新 state，不直接修改原对象。
  • dispatch(action) 描述“发生了什么”，reducer 决定“状态如何变化”。
