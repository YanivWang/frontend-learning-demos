条件渲染中的「短路求值」陷阱

在浏览器中操作下方交互演示；复习要点与面试答法见上方区块。

知识点要点

  • {falsyValue && <X/>} 只有当 falsyValue 为 false / null / undefined 时不渲染。
  • 反面教材：{arr.length && <X/>} 当 arr.length === 0 时短路返回 0，会被渲染成「0」文本节点。
  • 安全写法：{arr.length > 0 && <X/>} 或 {Boolean(arr.length) && <X/>}。
  • React 18 统一用 createRoot(...).render(...)，ReactDOM.render 已废弃。
