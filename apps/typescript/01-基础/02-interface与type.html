interface vs type

在浏览器中打开本页，结合下方演示区观察效果；要点与面试答法见上方复习区。

知识点要点

  • interface 描述对象形状，可声明合并（同名 interface 自动合并）。
  • type 可描述联合、交叉、元组、映射等更复杂类型。
  • 对象形状优先 interface；联合/工具类型常用 type。
  • 两者都支持 extends / 交叉（&）组合。
