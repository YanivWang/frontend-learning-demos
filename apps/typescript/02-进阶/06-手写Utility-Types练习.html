手写 Utility Types（运行时等价演示）

知识点要点

  • Partial&lt;T&gt;：所有属性可选。
  • Pick&lt;T, K&gt; / Omit&lt;T, K&gt;：挑选或排除键。
  • 实现思路：映射类型 + keyof + in。
  • 面试：理解内置工具类型比背 API 更重要。

面试考点

  • Partial&lt？ Partial&lt;T&gt;：所有属性可选
  • 手写 Utility Types 练习的核心概念是什么？ Partial&lt;T&gt;：所有属性可选；Pick&lt;T, K&gt; / Omit&lt;T, K&gt;：挑选或排除键
  • 手写 Utility Types 练习在实际项目里怎么用？ Partial&lt;T&gt;：所有属性可选；Pick&lt;T, K&gt; / Omit&lt;T, K&gt;：挑选或排除键
type MyPartial<T> = { [K in keyof T]?: T[K] };
type MyPick<T, K extends keyof T> = { [P in K]: T[P] };
type MyOmit<T, K extends keyof T> = Pick<T, Exclude<keyof T, K>>;
