React 合成事件机制

在浏览器中操作下方交互演示；复习要点与面试答法见上方区块。

知识点要点

  • React 事件是 SyntheticEvent，屏蔽浏览器差异。
  • React 事件名使用小驼峰，如 onClick、onChange。
  • React 17+ 事件委托从 document 调整到 root 容器。
  • 阻止默认行为使用 event.preventDefault()，不要 return false。
