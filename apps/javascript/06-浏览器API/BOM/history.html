history pushState / replaceState 与 popstate

下方为 console.log 同步输出；也可打开 DevTools Console 查看。

知识点要点

  • pushState 在历史栈中追加一条记录，可后退；replaceState 替换当前条目。
  • state 对象在 popstate 事件中可通过 event.state 取回。
  • 前进/后退或 history.back/forward 触发 popstate；直接 pushState 不触发。
