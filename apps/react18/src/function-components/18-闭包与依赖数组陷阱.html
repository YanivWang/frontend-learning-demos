闭包与依赖数组陷阱

在浏览器中操作下方交互演示；复习要点与面试答法见上方区块。

知识点要点

  • 函数组件每次渲染都有自己的 state 快照和函数闭包。
  • effect / callback 使用了外部变量，就应该放入依赖数组。
  • 故意不放依赖会产生 stale closure，也就是读到旧状态。
  • 如果只需要基于旧状态更新，函数式 setState 可以减少依赖。
