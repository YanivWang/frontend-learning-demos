vue-router 路由重定向 与 alias 别名

在浏览器中操作下方交互演示；复习要点与面试答法见上方区块。

知识点要点

  • redirect: 命中后地址栏会被替换；URL 与匹配到的路由不再是 /a 而是 /b。
  • alias: 让 URL 保持原样，但实际渲染原路径对应的组件，常用于隐藏 UI 内部结构。
  • 路由对象支持 name 命名路由，便于 router.push({ name: 'toC' }) 调用。
