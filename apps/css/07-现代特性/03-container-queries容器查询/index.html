Container Queries — 按父容器宽度自适应

拖动左侧虚线框改变容器宽度。容器 ≥ 360px 时卡片由纵向堆叠变为横向排列（与浏览器视口无关）。

知识点要点

  • 媒体查询看视口宽度；容器查询看「父容器」宽度，适合卡片/侧边栏等可复用组件。
  • 父元素设置 container-type: inline-size（或 size）并可选 container-name。
  • @container (min-width: 400px) 在容器够宽时切换布局，与 viewport 无关。
  • 面试对比：media query 全局断点 vs container query 组件自适配。
