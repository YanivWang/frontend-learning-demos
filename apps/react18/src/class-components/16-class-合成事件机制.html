class 组件中的 React 合成事件

在浏览器中操作下方交互演示；复习要点与面试答法见上方区块。

知识点要点

  • 事件对象是 SyntheticEvent，接口与原生事件类似。
  • event.nativeEvent 可访问底层原生事件。
  • 阻止默认行为用 preventDefault()；阻止冒泡用 stopPropagation()。
  • React 17+ 事件委托绑定在 root 容器而非 document。
