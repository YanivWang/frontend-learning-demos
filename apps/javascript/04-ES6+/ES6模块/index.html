ES Module import/export 静态导入与动态导入

下方为 console.log 同步输出；也可打开 DevTools Console 查看。

知识点要点

  • script type="module" 自动启用严格模式，import 必须是顶层语句。
  • 静态导入 import {...} from '...'，动态导入 import('...').then(...)。
  • 模块在浏览器中需要通过 http(s) 协议加载（直接 file:// 不行）。
  • script type="module" 自动启用严格模式，import 必须是顶层语句（补充）。
