Date 常用工具函数

JavaScript 原生 Date 用法简单但容易踩坑（月份从 0 开始；getDate 是日不是星期…）。

知识点要点

  • getMonth() 0～11；getDay() 0=周日；getDate() 是「日」不是星期。
  • new Date(y, m, 0) 中 m 为 1-based 月份时，day=0 表示上个月的最后一天，即当月月末。
  • 时间戳：Date.now()、getTime()；解析字符串推荐 ISO 8601。
  • 格式化：优先 Intl.DateTimeFormat 或 dayjs/date-fns，避免手写 pad。
  • Date 是可变对象，运算前注意 clone；跨时区业务用 UTC 或专用库。

面试考点

  • Date 常见坑？ 月份从 0 起；new Date('2024-01-01') 时区差异；直接改实例会 mutate。
  • 某月最后一天？ new Date(year, month, 0).getDate()（month 为下一月 1-based）或 dayjs 的 endOf('month')。
  • 时区处理？ 存储 UTC 时间戳；展示用 toLocaleString + timeZone；复杂场景用 Temporal（未来）或 dayjs timezone 插件。
