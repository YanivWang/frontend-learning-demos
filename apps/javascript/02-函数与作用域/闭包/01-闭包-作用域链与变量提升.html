闭包的作用域链与变量提升

请打开 DevTools Console 查看输出。

知识点要点

  • var 在脚本顶层 → window 属性；let/const → 脚本作用域，不挂 window。
  • 函数声明 + var 有声明提升；let/const 有暂时性死区。
  • 闭包：内层函数引用外层变量，会延长外层变量的生命周期。
  • var 在脚本顶层 → window 属性；let/const → 脚本作用域，不挂 window（补充）。
