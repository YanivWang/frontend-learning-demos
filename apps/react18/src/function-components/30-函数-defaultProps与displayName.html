defaultProps、参数默认值与 displayName

在浏览器中操作下方交互演示；复习要点与面试答法见上方区块。

知识点要点

  • 函数组件推荐 ES6 参数默认值: function Greeting({ name = "访客" })。
  • 仍可用 Component.defaultProps，但新代码更推荐解构默认值。
  • displayName 便于 React DevTools 调试；HOC 包装后尤其需要设置。
  • 对比 class 17：函数组件没有 static，displayName 直接挂函数名上。
