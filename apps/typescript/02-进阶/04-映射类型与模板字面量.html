映射类型与模板字面量类型

在浏览器中打开本页，结合下方演示区观察效果；要点与面试答法见上方复习区。

知识点要点

  • 映射类型遍历 keyof T，可以批量改变属性可选、只读或值类型。
  • 索引访问 T[K] 可根据字段名取得字段类型，是 Pick、Record 等工具的基础。
  • 模板字面量类型可拼接字符串字面量，适合事件名、接口路径、CSS 变量名。
  • as 重映射键名可把对象字段转换成 onXxx / getXxx 这类 API。

type User = { id: number; name: string; active: boolean };

type FieldGetter<T> = {
  [K in keyof T as `get${Capitalize<string & K>}`]: () => T[K]
};

type UserEvent = `${keyof User & string}Changed`;
type NameType = User["name"];  // 索引访问：string
