函数声明的最小可运行示例

请打开 DevTools Console 查看输出。

知识点要点

  • 函数声明 function fn() {} 会被整体提升，声明前即可调用。
  • return 将值传回调用处；无 return 或空 return 时返回 undefined。
  • 形参是局部变量；实参个数可与形参不一致，多出的参数在普通函数中通过 arguments 访问。
  • 函数是一等公民，可赋值给变量、作为参数传递、作为返回值（详见 函数进阶）。

面试考点

  • 函数声明 vs 函数表达式？ 声明会整体提升；表达式只是变量赋值，遵循 var/let 的提升规则。表达式在赋值前不可调用。
  • 函数声明提升 vs var 提升？ 同作用域内函数声明优先于 var；若同名，函数声明覆盖 var 的 undefined 初始值。
  • 没有 return 返回什么？ 显式或隐式返回 undefined。注意 return 后的换行不会自动插入分号导致 ASI 陷阱（见进阶题）。
