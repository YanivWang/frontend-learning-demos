浏览器渲染与性能

请打开 DevTools Console 查看输出。

知识点要点

  • 关键渲染路径：HTML → DOM，CSS → CSSOM，然后生成 Render Tree。
  • Layout 计算位置尺寸，Paint 绘制像素，Composite 合成图层。
  • 重排 reflow：几何信息变化；重绘 repaint：视觉样式变化但布局不变；合成 composite：transform/opacity 等。
  • 批量 DOM 操作、读写分离、transform/opacity 动画可以减少性能压力。

面试考点

  • 重排 vs 重绘 vs 合成？ 重排最贵（改布局）；重绘次之（改颜色）；合成最轻（只改 transform/opacity 图层）。
  • 如何减少重排？ DocumentFragment 批量插入、避免 layout thrashing（读写分离）、离线修改 DOM。
  • 动画优化？ 优先 transform/opacity 触发合成；节流用 requestAnimationFrame。
