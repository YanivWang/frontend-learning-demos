字符串截取三兄弟 + 常用方法

三者都不会修改原字符串，但参数语义差异很大，是高频面试坑点。请打开 DevTools Console 查看输出。

知识点要点

  • slice(start, end)：推荐。支持负数索引；end 不含；start>end 返回空串。
  • substring(start, end)：负数当 0；start>end 自动交换起止。
  • substr(start, length)：已废弃，第二参数是长度而非 end。
  • 字符串是原始类型，方法都不改原串，返回新串；new String() 是包装对象。
  • 常用：includes、startsWith、endsWith、indexOf、replace、split、trim、padStart。

面试考点

  • slice / substring / substr 区别？ slice 最通用（负索引）；substring 交换起止；substr 按长度且已废弃。新项目只用 slice。
  • 字符串不可变？ 原始 string 不可改；修改操作返回新串。大量拼接用数组 join 或模板字符串，避免 + 循环。
  • 包含子串？ ES6 includes；需位置用 indexOf !== -1；正则用 test / match。
