extends 继承与 super 调用

下方为 console.log 同步输出；也可打开 DevTools Console 查看。

知识点要点

  • 派生类的 constructor 必须先调用 super() 才能使用 this。
  • 实例方法挂在 prototype 上，所有实例共享；实例字段每个实例各有一份。
  • 易错点：在实例方法中读私有字段要用 this.#field，不能写死成具体实例名。
