await 表达式与异常处理

请打开 DevTools Console 查看输出。

知识点要点

  • await 后面可以是任意表达式：非 Promise 会被 Promise.resolve 包装。
  • async 函数 return 的值 → 作为 then(data) 回调的参数。
  • 任一 await 的 Promise 变 reject，整个函数立刻中断；用 try/catch 捕获。
