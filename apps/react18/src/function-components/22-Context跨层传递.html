Context 跨层传递

在浏览器中操作下方交互演示；复习要点与面试答法见上方区块。

知识点要点

  • Context 适合主题、语言、当前登录用户等跨层共享数据。
  • Provider 提供 value，useContext 读取最近的上层 Provider。
  • Context 不是全局状态管理银弹；频繁变化的大状态可能导致较大范围重渲染。
