通过手动置 null 释放闭包引用

请打开 DevTools Console 查看输出。

知识点要点

  • inner 持有 outer 的局部变量 bigData，会阻止其回收。
  • 把 inner 置为 null 后，闭包链断裂，GC 可回收。
  • 用 DevTools 在置 null 前后做 Heap snapshot 对比可直观看到差异。
