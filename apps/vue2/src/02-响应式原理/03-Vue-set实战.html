Vue.$set 与 Object.assign 实战

在浏览器中操作下方交互演示；复习要点与面试答法见上方区块。

知识点要点

  • Vue2 不能检测「新增属性」与「数组下标赋值」，需要 Vue.set / this.$set 触发响应。
  • Object.assign(target, source) 原地合并到已有响应式对象（如 this.person）仍保持响应式。
  • 对需要整体替换的模型（如 this.dog），用 Object.assign({}, target, source) 生成新对象再赋值。
  • 演示 defineProperty 访问器：每次访问 deerObj.weight 都会触发 set/get。
