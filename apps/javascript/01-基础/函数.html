函数声明、参数与返回值基础

请打开 DevTools Console 查看输出。

知识点要点

  • 函数声明 function fn() {} 会被整体提升，声明前即可调用。
  • return 将值传回调用处；无 return 或空 return 时返回 undefined。
  • 形参是局部变量；实参个数可与形参不一致，多出的参数在普通函数中通过 arguments 访问。
  • 函数是一等公民，可赋值给变量、作为参数传递、作为返回值（详见 函数进阶）。

面试考点

  • 函数声明 vs 函数表达式？ 结论：函数声明在同作用域内可先调用，函数表达式要等赋值执行后才能调用。原因是声明会整体初始化为函数对象，而表达式遵循变量声明和赋值的执行顺序；项目里需要条件创建或传递回调时常用表达式。
  • 函数声明提升 vs var 提升？ 结论：函数声明提升后直接可调用，var 只提升声明并初始化为 undefined。同名时还要看后续赋值语句是否覆盖；复杂场景建议避免同名声明。
  • 没有 return 返回什么？ 结论：无 return 或空 return 都返回 undefined。原因是函数调用必须产生一个完成值；链式 API 或计算函数要显式返回，事件处理器这类副作用函数可以不返回。
  • 形参与实参数量不一致怎么办？ 结论：少传的形参是 undefined，多传的实参普通函数可从 arguments 读取。现代代码更推荐默认参数和剩余参数，让边界更清楚。
