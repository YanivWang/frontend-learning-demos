let + for 循环的块作用域闭包

请打开 DevTools Console 查看输出。

知识点要点

  • let 让每次循环迭代创建独立的块作用域，i 各自独立。
  • 输出 0..4；如果换成 var 则全是 5（共享同一个 i）。
  • 现代代码优先用 let/const 取代 var。
  • let 让每次循环迭代创建独立的块作用域，i 各自独立（补充）。
