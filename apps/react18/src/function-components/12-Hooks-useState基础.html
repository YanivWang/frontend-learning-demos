useState 基础与函数式更新

在浏览器中操作下方交互演示；复习要点与面试答法见上方区块。

知识点要点

  • Hooks 只能在函数组件顶层调用，不能写在 if / for / 普通函数里。
  • useState 返回 [state, setState]，修改状态必须调用 setState。
  • 当下一次状态依赖上一次状态时，优先使用函数式更新 setCount(prev => prev + 1)。
  • React 状态更新是异步调度的，setState 后立刻读取旧值是常见面试坑。
