合成层与 GPU 加速

在浏览器中打开本页，结合下方演示区观察效果；要点与面试答法见上方复习区。

知识点要点

  • transform / opacity 动画常走合成层，不触发布局。
  • will-change 提示浏览器提前优化，应短期使用。
  • 过多合成层会占用显存，需权衡。
