@keyframes 基础旋转动画

在浏览器中打开本页，结合下方演示区观察效果；要点与面试答法见上方复习区。

知识点要点

  • @keyframes name { 0% {...} 50% {...} 100% {...} } 定义多段关键帧。
  • animation: name duration timing-function iteration-count。
  • linear 等速；infinite 无限循环；transform 用 GPU 加速性能好。
