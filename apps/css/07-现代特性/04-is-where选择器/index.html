:is() 与 :where()

在浏览器中打开本页，结合下方演示区观察效果；要点与面试答法见上方复习区。

知识点要点

  • :is() 特异性 = 参数中最具体选择器的特异性。
  • :where() 特异性恒为 0，适合重置样式而不抬高优先级。
  • 简化复杂选择器列表，避免重复写父级。
  • 与 :not() 组合可表达「排除某些子元素」。

面试考点

  • 简化复杂选择器列表？ 简化复杂选择器列表，避免重复写父级

  • :is() 与 :where() 及特异性在实际项目里怎么用？ :is() 特异性 = 参数中最具体选择器的特异性

:is(h2, h3, strong) 等价于分别写三条规则；:where(p, li) 特异性为 0，易被后续规则覆盖。
