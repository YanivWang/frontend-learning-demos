高阶组件 HOC 与 render props

在浏览器中操作下方交互演示；复习要点与面试答法见上方区块。

知识点要点

  • HOC: 函数接收组件，返回增强后的新组件，不修改原组件。
  • render props: 组件接收函数 prop，由父决定渲染什么。
  • Hooks 时代很多 HOC/render props 场景可用自定义 Hook 替代。
  • 对比 class 18：函数组件同样可写 HOC，但面试常问与 Hook 的取舍。
