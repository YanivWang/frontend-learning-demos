下方为 console.log 同步输出；也可打开 DevTools Console 查看。

知识点要点

  • 类初始化阶段执行（不是 new 时），按声明顺序同步执行。
  • 块内的 this 指向「类构造函数对象」本身，可访问私有静态字段。
  • 只能引用「声明在自己上方」的字段；引用下方字段拿到 undefined。
