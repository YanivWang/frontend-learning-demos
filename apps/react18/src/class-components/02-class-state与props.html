class 组件的 state 与 props 区别 + 事件 this 绑定

在浏览器中操作下方交互演示；复习要点与面试答法见上方区块。

知识点要点

  • state 只能在 constructor 里直接赋值，其他地方必须用 setState。
  • props 是只读的；想通知父组件改值需通过父级传入的回调。
  • 事件函数若想拿到组件实例 this，可在 JSX 用箭头函数、constructor 里 bind、或类字段箭头函数（详见 13）。
  • onClick={console.log(this)} 会在 render 阶段立即执行，应包成箭头函数。
  • render 无参数；this.state 中无 id 字段时 <div id={this.state.id}> 为 undefined。
