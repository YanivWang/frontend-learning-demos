闭包导致的内存泄漏与修复手段

请打开 DevTools Console 查看输出。

知识点要点

  • 闭包持有外层变量引用，外层函数执行完毕这些变量也不会被回收。
  • 解决：调用后手动置 null、或避免持久持有大对象的闭包。
  • 配合 use strict 杜绝意外全局变量；定时器/事件监听记得清理。
