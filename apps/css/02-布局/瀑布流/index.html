瀑布流布局（CSS column-count，无需 JS）

在浏览器中打开本页，结合下方演示区观察效果；要点与面试答法见上方复习区。

知识点要点

  • column-count 把容器分成 N 列；column-gap 控制列间距。
  • break-inside: avoid 防止单个 item 被强行截断到下一列。
  • 配合 @media 在窄屏减少列数；缺点：从左到右逐列填充，与传统 JS 算法略不同。
