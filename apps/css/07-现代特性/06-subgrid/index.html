subgrid 卡片对齐

在浏览器中打开本页，结合下方演示区观察效果；要点与面试答法见上方复习区。

知识点要点

  • subgrid 让嵌套 grid 继承父 grid 的轨道尺寸，列对齐跨卡片。
  • grid-template-columns: subgrid 用于列；行同理 subgrid / subgrid rows。
  • 适合卡片列表标题/正文/按钮底对齐。
  • 不支持时降级为普通 grid（布局仍可用，对齐略差）。
