IIFE（Immediately Invoked Function Expression）

定义后立刻调用的函数。常用于：① 创建独立作用域，避免污染全局；② 模块的早期实现方式。

核心：函数表达式 + 后面的 ()

解析器看到 function 开头时，会优先把它当成函数声明， 所以必须用括号或运算符把它"逼成"函数表达式。

2. 立即传参：把外部变量"快照"到函数内

3. 经典面试题：用 IIFE 修复 var + for 闭包

4. 模块化的雏形：揭示模块模式（Revealing Module Pattern）
