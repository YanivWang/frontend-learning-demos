hash 路由的基本演示

请打开 DevTools Console 查看输出。

知识点要点

  • location.hash 修改不会触发整页刷新，是 hash 路由的基础。
  • 监听 hashchange 即可在 SPA 中实现「自定义导航」。
  • location.replace 会替换当前历史栈条目，不会留下「后退」记录。
