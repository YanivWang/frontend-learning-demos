Proxy 的局限性

下方为 console.log 同步输出；也可打开 DevTools Console 查看。

知识点要点

  • Proxy 对依赖「内部槽位」的方法不透明，例如 Date.getDate 调用会丢失内部 this。
  • 解决：在 get 拦截中显式 bind 原对象，或对方法做手动转发。
  • instanceof 判断不一定可靠：Date 的内部检测会失败。
  • Proxy 对依赖「内部槽位」的方法不透明，例如 Date.getDate 调用会丢失内部 this（补充）。
