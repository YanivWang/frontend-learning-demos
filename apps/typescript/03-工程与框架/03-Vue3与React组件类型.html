Vue3 / React 组件类型

在浏览器中打开本页，结合下方演示区观察效果；要点与面试答法见上方复习区。

知识点要点

  • Vue3 常用 defineProps<T>() / defineEmits<T>() 描述组件输入输出。
  • React 组件常直接给函数参数标 Props；React.FC<Props> 可用但不是必须。
  • 组件 Props 类型要表达业务约束，不只是把所有字段写成 string。
  • 面试常问 TS 如何提升组件边界、事件类型和表单数据可靠性。
