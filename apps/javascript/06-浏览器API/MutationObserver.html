MutationObserver 监听 DOM 变化

请打开 DevTools Console 查看输出。

知识点要点

  • 回调在微任务队列中批量执行，多次变更会合并触发一次。
  • config 中的 subtree:true 会扩大监听到所有后代，开销较大。
  • 易错点：在回调中再修改 DOM 容易死循环；observer.disconnect 用完要解绑。
