序列帧动画（精灵图）

在浏览器中打开本页，结合下方演示区观察效果；要点与面试答法见上方复习区。

知识点要点

  • 精灵图（sprite）横向拼接多帧，background-size 设为完整长度。
  • steps(5, end) 把 5 帧拆成 5 步离散跳变，配合 keyframes 从 0 到 -2000px。
  • animation-play-state: paused/running 实现暂停/播放控制。
