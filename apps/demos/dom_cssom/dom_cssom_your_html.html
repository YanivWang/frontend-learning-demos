DOM Tree
  • <!DOCTYPE html>
    • document
      • <html lang="en">
        • <head>
          • <meta charset="UTF-8">自闭合
          • <meta name="viewport" …>自闭合
          • <title>
            • "Document"Text
          • <style>→ 触发 CSSOM 构建
            • "h1{color:blue} h3{color:red}"Text（原始 CSS 字符串）
        • <body>
          • <h1>
            • "DOM"Text
          • <h3>
            • "DOM讲解"Text
          • <script>内容为空
CSSOM Tree
  • CSSStyleSheetdocument.styleSheets[0]
    • CSSStyleRulerules[0]
      • .selectorText → "h1"
      • .style CSSStyleDeclaration
        • color : blue
    • CSSStyleRulerules[1]
      • .selectorText → "h3"
      • .style CSSStyleDeclaration
        • color : red
在控制台可以验证：
document.styleSheets[0].cssRules[0].selectorText
→ "h1"

document.styleSheets[0].cssRules[0].style.color
→ "blue"

document.styleSheets[0].cssRules[1].selectorText
→ "h3"
浏览器解析完两棵树后，将 h1（匹配 color:blue）和 h3（匹配 color:red）的样式附加到对应 DOM 节点，合成 Render Tree，再 Layout → Paint。
