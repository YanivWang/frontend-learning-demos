class 组件 constructor 必须调用 super()

在浏览器中操作下方交互演示；复习要点与面试答法见上方区块。

知识点要点

  • class extends React.Component 时，一旦显式写了 constructor，必须先 super(props)。
  • 否则会抛 ReferenceError；constructor 内不能在 super() 之前访问 this。
  • class 组件必须实现 render()，否则抛 "No render method found"。
  • class extends React.Component 时，一旦显式写了 constructor，必须先 super(props)（补充）。
