知识点要点

  • addEventListener 第三个参数 true 表示捕获阶段监听。
  • stopPropagation 阻止事件继续传播；stopImmediatePropagation 还会阻止同元素后续监听器。
  • 事件委托把子元素点击统一交给父元素处理，适合动态列表。
