浏览器存储方案对比

方案 容量 生命周期 是否随请求发送 API 特点 常见用途
Cookie 约 4KB 由 Expires / Max-Age 控制；默认会话级 会，同源请求自动带上 字符串 API；可配置 Domain / Path / SameSite / Secure / HttpOnly 登录态、少量服务端需要读取的信息
sessionStorage 通常约 5MB 当前标签页关闭即清除 不会 同步 key-value API 页面临时状态、表单草稿
localStorage 通常约 5MB 手动清除前一直存在 不会 同步 key-value API，大量读写可能阻塞主线程 主题、偏好设置、小型缓存
IndexedDB 远大于 localStorage，浏览器按磁盘和策略限制 手动删除或浏览器清理前存在 不会 异步事务型数据库 API，可存对象和 Blob 离线数据、大文件、复杂结构缓存
