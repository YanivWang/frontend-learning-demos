Service Worker 注册

知识点要点

  • SW 独立线程，可拦截 fetch、做离线缓存（PWA 核心能力）。
  • 必须在 HTTPS / localhost 下注册，且单独提供 sw.js。
  • 注册晚于 load 事件可避免阻塞首屏关键资源。
