ref 作为 prop 传递（React 19）

在浏览器中操作下方交互演示；复习要点与面试答法见上方区块。

知识点要点

  • React 19 中函数组件可直接接收名为 ref 的 prop，无需 forwardRef 包裹。
  • 父组件 useRef() 后把 ref 传给子组件，子组件绑定到原生 DOM。
  • 类组件、第三方库迁移可逐步去掉 forwardRef 样板代码。
  • 命令式 API 仍推荐优先声明式；ref 用于聚焦、滚动、测量等。
