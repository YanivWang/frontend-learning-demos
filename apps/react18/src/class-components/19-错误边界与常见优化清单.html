在浏览器中操作下方交互演示；复习要点与面试答法见上方区块。

知识点要点

  • Error Boundary 只能用 class 组件实现，捕获子组件渲染、生命周期、构造函数中的错误。
  • 它不能捕获事件处理、异步回调、服务端渲染、错误边界自身的错误。
  • 常见优化: 合理 key、memo、useMemo/useCallback、懒加载、虚拟列表、状态下沉或拆分 Context。
