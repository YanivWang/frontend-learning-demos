列表 key 与 diff

在浏览器中操作下方交互演示；复习要点与面试答法见上方区块。

知识点要点

  • key 帮助 React 在同级列表中识别元素身份。
  • key 不会作为 props 传给组件。
  • 稳定唯一的业务 id 通常优于 index。
  • 用 index 作为 key 时，列表插入、删除、排序可能导致状态错位。
