this 的动态绑定

下方为 console.log 同步输出；也可打开 DevTools Console 查看。

知识点要点

  • 函数定义时 this 不确定，运行时由「调用方」决定。
  • window.a() 中 this 指 window；window.a()() 第二次调用以 undefined/window 收尾。
  • this 永远指向「最后调用它」的对象。
  • 函数定义时 this 不确定，运行时由「调用方」决定（补充）。
