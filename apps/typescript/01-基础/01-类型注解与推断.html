类型注解与推断

知识点要点

  • TS 在 JS 上增加静态类型；变量可显式注解 let n: number = 1。
  • 未注解时编译器会推断类型；推断结果在 IDE / tsc 中生效，浏览器运行仍是 JS。
  • 基本类型：string / number / boolean / null / undefined / bigint / symbol。
  • 面试常问：any 放弃检查；unknown 更安全，使用前须收窄。
