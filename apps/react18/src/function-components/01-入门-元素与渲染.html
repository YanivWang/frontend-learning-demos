React 元素与渲染（React 18 createRoot）

在浏览器中操作下方交互演示；复习要点与面试答法见上方区块。

知识点要点

  • JSX 由 Babel 转译为 React.createElement(type, props, ...children)。
  • createElement 返回一个普通对象（虚拟 DOM），由 ReactDOM 负责实际渲染。
  • React 18 推荐用 ReactDOM.createRoot(container).render(...)，旧 ReactDOM.render 已废弃。
  • JSX 中引用变量前须确保已定义；未定义的 {ele1} 会 ReferenceError。
