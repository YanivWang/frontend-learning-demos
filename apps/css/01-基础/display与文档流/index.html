知识点要点

  • block 独占一行，可设置宽高；inline 不独占一行，宽高通常不生效。
  • inline-block 不独占一行，但可以设置宽高，常用于文字旁的小块元素。
  • display:none 不占位；visibility:hidden 占位但不可见。
