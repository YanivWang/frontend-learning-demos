JSON.stringify 进阶用法

请打开 DevTools Console 查看输出。

知识点要点

  • JSON.stringify(value, replacer?, space?) 仅支持 object、array、string、number、boolean、null。
  • replacer 为数组时作属性白名单；为函数时按 (key, value) 返回值，返回 undefined 则忽略该属性。
  • undefined、function、Symbol 在对象中会被忽略；数组中变成 null；Date 变 ISO 字符串。
  • 循环引用会抛 TypeError；NaN/Infinity 变成 null。
  • JSON.parse 不能解析 trailing comma、单引号等非标准 JSON。

面试考点

  • JSON.stringify 限制？ 不支持 function、undefined、Symbol、循环引用；Date 变字符串；Map/Set 需自行转换。
  • replacer 用法？ 数组过滤字段；函数自定义序列化（如 Date 转时间戳）。
  • JSON 深拷贝？ JSON.parse(JSON.stringify(obj)) 简单但丢函数/Date/undefined/循环引用。完整方案见手写深拷贝。
