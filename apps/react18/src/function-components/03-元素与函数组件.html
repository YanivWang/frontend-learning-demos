React 元素、JSX 嵌入表达式与 createRoot

在浏览器中操作下方交互演示；复习要点与面试答法见上方区块。

知识点要点

  • 文件名保留历史命名「元素与函数组件」；函数组件系统讲解见 05-函数组件-props.html 起。
  • JSX 中 {expression} 只能是表达式；函数引用 {fn} 不会执行，需写 {fn()} 才能得到返回值。
  • 元素是描述 UI 的轻量对象；createRoot(container).render(element) 触发虚拟 DOM → diff → 真实 DOM。
  • React 18 统一使用 createRoot，旧 ReactDOM.render 已废弃。
