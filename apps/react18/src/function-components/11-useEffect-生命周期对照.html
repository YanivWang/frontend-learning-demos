useEffect 与 class 生命周期对照

在浏览器中操作下方交互演示；复习要点与面试答法见上方区块。

知识点要点

  • componentDidMount ≈ useEffect(fn, [])。
  • componentDidUpdate ≈ useEffect(fn, [deps]) 在 deps 变化时的行为（注意：带 deps 的 effect 在 mount 时也会执行，与 componentDidUpdate 不同）。
  • componentWillUnmount ≈ useEffect 返回的 cleanup。
  • 没有一一对应关系：一个 useEffect 可组合多种时机，应按「副作用语义」拆分。
  • StrictMode 下（见 26）开发环境会 mount → unmount → mount，effect 与 cleanup 各执行两次；生产环境不受影响。
