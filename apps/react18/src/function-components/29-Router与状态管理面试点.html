Router 与状态管理常见面试点

在浏览器中操作下方交互演示；复习要点与面试答法见上方区块。

知识点要点

  • React Router 负责 URL 与页面组件的映射，常见概念有 BrowserRouter、Routes、Route、Link、useNavigate、useParams。
  • 状态管理要先区分本地状态、跨组件状态、服务端缓存状态。
  • Redux/Zustand/Context 不是互相完全替代，选择取决于规模、协作和调试需求。
  • 本文件不引入第三方库，用最小 demo 模拟路由与 store 思想。
