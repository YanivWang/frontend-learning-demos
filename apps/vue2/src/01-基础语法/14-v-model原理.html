知识点要点

  • 表单元素上的 v-model 本质是 value 绑定 + input/change 事件。
  • text/textarea 默认使用 value + input；checkbox/radio/select 有各自的表单语义。
  • 自定义组件上的 v-model 默认等价于 :value="x" @input="x = $event"。
  • 表单元素上的 v-model 本质是 value 绑定 + input/change 事件（补充）。
