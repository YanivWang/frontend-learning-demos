类的字段、方法与原型对象关系

下方为 console.log 同步输出；也可打开 DevTools Console 查看。

知识点要点

  • 公有/私有实例字段「每实例一份」；实例方法挂在 prototype 上「所有实例共享」。
  • 在 constructor 里给 this 赋方法，每个实例会各持有一份，浪费内存。
  • 私有字段只能在类体外通过类内部代码访问，且只能用 this.#field。
