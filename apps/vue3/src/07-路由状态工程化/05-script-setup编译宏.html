在浏览器中操作下方交互演示；复习要点与面试答法见上方区块。

知识点要点

  • defineProps / defineEmits / defineExpose 是编译宏，无需 import，只在 SFC 编译期生效。
  • defineModel（3.4+）简化组件 v-model，等价 props + emit update:modelValue。
  • withDefaults(defineProps&lt;...&gt;(), { ... }) 为 TS props 提供默认值。
  • 宏只能在 .vue 的 script setup 顶层使用；本 demo 用等价运行时 API 演示行为。
