React 19 新特性概览

在浏览器中操作下方交互演示；复习要点与面试答法见上方区块。

知识点要点

  • React 19：Actions（useTransition 内 async）、useActionState、useOptimistic。
  • use() 读取 Promise / Context，可在条件分支中调用（规则变化）。
  • ref 作为 prop 传递，不再需要 forwardRef（逐步迁移）。
  • 本页使用 React 18.3.1 模拟 Action 思路；可运行 React 19 demo 见 apps/react19/。
