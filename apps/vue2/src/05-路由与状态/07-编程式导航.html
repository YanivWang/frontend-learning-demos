编程式导航（push / replace / go / 命名路由）

在浏览器中操作下方交互演示；复习要点与面试答法见上方区块。

知识点要点

  • $router.push 向 history 栈追加记录，浏览器后退可返回。
  • $router.replace 替换当前记录，不入栈。
  • $router.go(n) / back() / forward() 在历史栈中移动。
  • 命名路由：router.push({ name: 'user', params: { id: '1' }, query: { tab: 'profile' } })。
