RSC 与 Server Components 面试边界

在浏览器中操作下方交互演示；复习要点与面试答法见上方区块。

知识点要点

  • Server Component：在服务端运行，默认不打包到客户端，可直接访问 DB/文件系统。
  • Client Component：文件顶行 'use client'，可使用 Hooks 与浏览器 API。
  • Server Actions：服务端函数，经编译后由客户端 form action 调用。
  • 本仓库为静态 HTML demo，RSC 需在 Next.js 等框架中体验；面试答清边界即可。
