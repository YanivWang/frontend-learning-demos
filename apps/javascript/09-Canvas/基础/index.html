知识点要点

  • 流程：getContext('2d') → 设置样式 → 路径 / 形状 → fill / stroke。
  • 同一条 path 内的所有线段会一起 fill/stroke；使用 beginPath 隔离不同路径。
  • 渐变、文字、圆角矩形等都基于同一套 ctx API。
