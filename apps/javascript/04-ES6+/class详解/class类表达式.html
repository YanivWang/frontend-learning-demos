class 表达式（匿名/具名）

请打开 DevTools Console 查看输出。

知识点要点

  • 类表达式可以是匿名或具名，具名时类名只在类内部可见。
  • 与函数表达式类似，类表达式不会进行声明提升。
  • 没有 extends 的类，constructor 内不能调用 super()。
