原型链 / 组合继承

请打开 DevTools Console 查看输出。

知识点要点

  • 组合继承：子构造函数中 Parent.call(this, ...) 借实例属性 + 原型链接 Object.create(Parent.prototype)。
  • 组合继承后须重置 Child.prototype.constructor = Child，否则 constructor 指向父类。
  • 注意：父类静态属性不会通过原型链继承。

每个对象都通过内部 [[Prototype]]（即 __proto__）指向其构造函数的 prototype，逐级向上形成原型链。
