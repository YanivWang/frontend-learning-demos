IIFE（Immediately Invoked Function Expression）

请打开 DevTools Console 查看输出。

知识点要点

  • IIFE = 函数表达式 + 紧跟的 ()；function 开头会被当声明，需括号或一元运算符逼成表达式。
  • 常见写法：(function(){})()、!function(){}()、具名 IIFE、带参 IIFE。
  • 可返回对象/函数，实现揭示模块模式（Revealing Module），隐藏私有变量。
  • 经典题：用 IIFE 为每次循环创建独立作用域，修复 var+for 闭包；现代用 let。
  • 现代 ES Module 取代 IIFE 做模块隔离；IIFE 仍见于 legacy 代码和需要立即执行的场景。

面试考点

  • IIFE 是什么？ 定义后立即执行的函数表达式，创建独立作用域，避免变量挂到全局。
  • var+for 闭包怎么修？ IIFE 传 i：(function(j){ setTimeout(...) })(i)；或直接用 let。
  • 和模块化的关系？ IIFE 是 ES5 时代的模块雏形；ES6 import/export 是标准方案。UMD 等仍可能包一层 IIFE。
