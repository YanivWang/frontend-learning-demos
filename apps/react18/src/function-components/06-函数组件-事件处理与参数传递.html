函数组件事件处理与参数传递

在浏览器中操作下方交互演示；复习要点与面试答法见上方区块。

知识点要点

  • 函数组件没有 this，事件处理函数直接定义即可，无需 bind。
  • 需要传参时用箭头函数包装: onClick={() => handle(id)}，不要写 handle(id) 立即执行。
  • 事件对象 e 作为第一个参数传入；需要同时传参时写 (e) => handle(e, id)。
  • 父组件通过 props 传入回调，子组件在事件里调用；不是 DOM 原生事件（对比 class 13 的 this 绑定）。
