知识点要点

  • 直接递归调用同步函数会爆栈；requestAnimationFrame 把下次调用挂到下一帧。
  • rAF 与浏览器刷新率同步（一般 60Hz），适合做动画。
  • 想要停止：保存返回的 id 并 cancelAnimationFrame(id)。
