知识点要点

  • 水平垂直居中优先用 flex/grid，老方案可用 absolute + transform。
  • 两栏布局常用 flex，侧边固定，主区域 flex:1。
  • 粘性 footer 可用 body min-height:100vh + flex column + main flex:1。
  • 水平垂直居中优先用 flex/grid，老方案可用 absolute + transform（补充）。
