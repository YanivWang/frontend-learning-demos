测试 localforage 最大可写容量

请打开 DevTools Console 查看输出。

知识点要点

  • 不同浏览器、不同后端（IndexedDB/WebSQL/localStorage）限制差异巨大。
  • 一次写入巨大数组可触发 QuotaExceededError，需要 try/catch。
  • JS 字符串按 UTF-16 存储，一个字符约 2 字节（emoji 等更多）。
