generator 基础：yield 与 next

请打开 DevTools Console 查看输出。

知识点要点

  • function* 定义生成器，调用后返回「迭代器对象」，函数体不会立即执行。
  • 每次 .next() 推进到下一个 yield，返回 { value, done }。
  • 普通对象没有默认迭代器；Array / Map / Set / String 等内置可迭代对象有 Symbol.iterator。
