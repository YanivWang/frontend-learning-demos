单行省略、行内图片对齐与替换元素适配

下方依次演示单行省略、图片基线空隙，以及用 object-fit 让图片裁切填满固定尺寸盒子。

知识点要点

  • 单行省略的条件是容器有可计算宽度，文本不换行，溢出被裁掉，最后由 text-overflow: ellipsis 显示省略号；少任一条件都可能只溢出或换行。
  • vertical-align 只影响行内、行内块和表格单元格的对齐。本页图片默认参与行内格式化，会按基线对齐而留下底部空隙；可用 vertical-align: middle 或把图片改为 display: block 处理。
  • object-fit 控制替换元素内容如何填充自身盒子，常见值有 cover、contain、fill；它不作用于普通 div 背景。

面试考点

  • 单行省略为什么必须同时设置三类属性？ 结论是省略号不是单靠 text-overflow 生成的。原因是浏览器只有在文本不换行、溢出被隐藏且容器发生行内方向溢出时，才会用省略号表示裁剪。项目里标题、表格单元格和面包屑都要先确认容器宽度是否明确。
  • img 底部空隙和 vertical-align 有什么关系？ 结论是底部空隙来自图片作为行内替换元素参与基线对齐。原因是行盒会为文字下伸部预留空间，图片默认贴基线而不是贴容器底边。项目里可以设 display: block 让图片脱离行内排版，或按需求用 vertical-align 调整。
  • object-fit 和 background-size 如何区分？ 结论是 object-fit 管替换元素内容，background-size 管背景图。原因是它们作用对象不同，不能互相替代。比如商品图用 img + object-fit 保留语义和 alt，装饰背景才用 background。
